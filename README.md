# The Wind Waker HD Randomizer Auto Tracker Wii U Plugin

A Wii U Aroma Homebrew Plugin that sends TCP packets over the network to your PC for the TWWHDR Tracker to receive and auto track items for you

## Building

Requirements:
- WUT
- WUPS
- Devkitpro

run `make` in the terminal of the directory of the makefile
